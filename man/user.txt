Usage: py gitus user <action> [options]

Actions:
  add        Creates a new user in activelist
  remove     Removes a existing user from activelist
  stash      List all users in activelist
  use        Selects a user to be the Git User

Options:
  add:
    [name]:  Name for the new user
    [email]: E-mail address for the new user
  remove:
    [ID]:    ID of the user to be deleted
  use:
    [ID]:    ID of the user to be selected as Git User
