Usage: py gitus <command> <action> [options]

Commands:
  user        Manage user-related operations
  list        Manages list-related operations

Additional Information:
  This script allows you to manage user data stored in a JSON file. Ensure that the 'users.json'
  file is present in the same directory as this script. The 'scripts' directory should contain the
  'rgen' module with the 'gen' function for generating user IDs.
